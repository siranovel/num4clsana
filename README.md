## install ##

From rubygems:  
~~~
    [sudo] gem install num4clsana
~~~

Install via Gemfile:  
~~~
source "https://rubygems.pkg.github.com/siranovel" do
    gem "num4clsana"
end
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4clsana.git  
    cd num4regana  
    gem build num4clsana.gemspec
    gem install num4clsana.gem
~~~
